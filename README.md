# sistema-armario-virtual
O Armário Virtual é um aplicativo voltado para o ambiente de lojas físicas, operado exclusivamente pelo vendedor, que transforma a experiência de compra por meio de provas digitais inteligentes. Após coletar as medidas do cliente e registrar seu consentimento, o vendedor captura uma foto e a IA gera um modelo corporal fiel ao biotipo da pessoa.
